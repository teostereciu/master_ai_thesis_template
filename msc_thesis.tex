        \chapter*{Abstract}
                \begin{adjustwidth}{4cm}{4cm}
                        Tool-use benchmarks assume that tools already exist. This thesis examines the step before: whether large language models (LLMs) can synthesize functional API tool servers from documentation, and whether those servers enable effective agent task completion. We introduce an end-to-end benchmark covering 14 REST APIs across five domains, in which a synthesis agent reads raw developer documentation and produces a complete MCP server --- a runnable, protocol-compliant collection of tools that an agent can discover and invoke directly, not merely individual function stubs --- a separate evaluation agent then attempts 13--25 natural-language tasks against it; and a third model judges each outcome.

                        We evaluate three synthesis conditions---with full documentation, without any documentation, and with documentation whose core parameter names have been deliberately renamed to conflict with training-time knowledge---across eight synthesis models spanning three capability tiers. Two metrics are reported: PASS (agent task success rate) and SYNTH (synthesis sufficiency, assessed by an LLM judge on the server's tool list without running the evaluation agent).

                        Our main findings are: (i)~synthesis is feasible for most APIs tested---eleven of fourteen achieve SYNTH~$\geq 75\%$ in the docs condition, with a frontier-tier median PASS of 83\%; (ii)~synthesized servers match or exceed human-authored reference implementations on GitHub and Zulip; (iii)~removing documentation does not degrade---and sometimes improves---agent performance on high-familiarity APIs, because model weights already encode correct implementations; (iv)~mutation adherence is near-zero for well-known public APIs (GitHub, Stripe, Shopify), confirming that parametric knowledge dominates in-context documentation for those APIs; and (v)~SYNTH and PASS are strongly correlated ($r = 0.937$, $p < 0.0001$, $n = 209$), making synthesis sufficiency a reliable and cheap proxy for downstream agent performance. A preliminary logit-lens probing study on an open-weight model localises parametric gravity to early-to-mid transformer layers, providing a mechanistic grounding for the behavioural findings.
                \end{adjustwidth}
